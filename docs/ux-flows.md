# UX Flows

## Onboarding

- Triggered if onboarding flag is false.
- Single screen: Welcome.

## Daily setup

- If no entry for today, present setup.
- If yesterday had unfinished focuses, show Continue/Fresh prompt.
- Must create 1-3 focuses.
- Each focus can optionally include a tag.

## Home

- Shows current day focuses with completion state.
- Tapping a focus toggles completion.
- Context menu allows editing focus title and tag.
- Weekly review card summarizes the last 7 days by tag.
- Streak milestones (7/14/30) trigger a share card.

## Settings

- Notification time picker + permission status.
- Toggles for midday/evening check-ins.
- Theme picker.
- Appearance mode selector.
