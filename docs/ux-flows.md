# UX Flows

## Onboarding

- Triggered if onboarding flag is false.
- Four-step carousel with skip/continue.

## Daily setup

- If no entry for today, show inline focus inputs on Home.
- If yesterday had unfinished focuses, show Continue/Fresh prompt to prefill.
- Must create 1-3 focuses.
- Each focus can optionally include a tag.

## Home

- Shows current day focuses with completion state.
- Inline editing for focus title and tag on Home.
- Weekly line shows a 7-day completion overview.
- Streak milestones (7/14/30) trigger a share card.

## Settings

- Bottom sheet from Home.
- Notification time picker + permission status.
- Toggles for midday/evening check-ins.
- Appearance mode selector.
